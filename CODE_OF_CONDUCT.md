# Community conduct

Mise welcomes rigorous, good-faith technical discussion. Be direct about code and
risk while remaining respectful toward the people doing the work.

Expected behavior:

- critique claims with reproducible evidence;
- distinguish a defect from a difference in taste;
- disclose conflicts, limitations, and AI assistance;
- protect private data and report security concerns privately;
- make space for contributors with different backgrounds and experience levels;
- accept correction and update the record when evidence changes.

Harassment, threats, discriminatory language, doxxing, sexualized attention, and
deliberate disruption are not acceptable in repository spaces. The maintainer may
edit, hide, lock, or remove contributions and participation that violate these
expectations.

Report conduct concerns privately to the repository owner through their GitHub
profile. Avoid putting sensitive personal details in a public issue.
